# 2D Physics Engine
Based on my [Verlet integration solver](https://github.com/wsandy1/rust-verlet-solver). _Currently trying to find a cool name... open an issue or contact me if you think of one..._

### Intended features
* Complex polygon collisions using the [hyperplane separating theorem](https://en.wikipedia.org/wiki/Hyperplane_separation_theorem)
* Mass and momentum
* Air resistance
* Friction
